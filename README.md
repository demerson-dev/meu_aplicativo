# 💪 FitMax

Aplicativo mobile desenvolvido para auxiliar usuários na criação, organização e acompanhamento de treinos físicos.
O objetivo do projeto é facilitar o planejamento de atividades físicas e permitir que o usuário acompanhe sua evolução de forma prática e estruturada.

SOBRE O PROJETO:

O FitMax é uma solução voltada para entusiastas do fitness que desejam ter mais controle sobre seus treinos. A aplicação permite que o usuário crie rotinas personalizadas, organize exercícios e acompanhe seu progresso ao longo do tempo.

FUNCIONALIDADES:

-Criar e gerenciar treinos personalizados;
-Organizar exercícios de forma estruturada;
-Acompanhar progresso e histórico de treinos;
-Acesso seguro e individual aos dados do usuário.
